# AI Operations Assistant

Proyecto de automatización e inteligencia artificial aplicada a operaciones y logística.

## Funcionalidades actuales

- Lectura de pedidos desde CSV
- Detección de incidencias logísticas
- Evaluación de riesgo operativo
- Preparado para integración con OpenAI API

## Tecnologías

- Python
- pandas
- OpenAI API
- VS Code

## Objetivo

Desarrollar un asistente inteligente capaz de:
- analizar incidencias,
- generar recomendaciones,
- automatizar reporting,
- y asistir en operaciones empresariales.
